# Random Background Color Changer

This project is a simple JavaScript-based web application that changes the background color randomly every second when the "Start" button is clicked. Clicking the "Stop" button halts the color changes.

## Features
- Randomly generates a hex color code.
- Changes the background color at one-second intervals.
- Provides start and stop functionality.
- Simple UI with improved CSS.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/Abhayrajsingh23/random-color-changer.git
   ```
2. Navigate to the project directory:
   ```sh
   cd random-color-changer
   ```
3. Open `index.html` in a browser.
4. Click **Start** to begin changing colors.
5. Click **Stop** to stop the color changes.
